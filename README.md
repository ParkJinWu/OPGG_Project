## 프로젝트 소개
### 아이디어 선정 이유
OP.GG는 라이엇이 제공하는 리그오브레전드의 데이터 분석을 통한 컨텐츠를 유저들에게 제공하는 사이트이다. 게임에 실용적인 정보 뿐 아니라 흥미에 초점을 둔 컨텐츠 또한 필요하다고 생각했다. MBTI검사와 같은 심리검사가 유행하였고, 이를 리그오브레전드 데이터에 접목하여 개인의 플레이 성향을 분석해보면 좋지 않을까 라는 생각에서 출발했다.
